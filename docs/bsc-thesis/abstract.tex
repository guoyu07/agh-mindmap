\begin{abstract}
	% What was done?
	This is a paper about project and implementation of mind-mapping solution for Android system with online real-time collaboration and offline synchronization.
	% Why was it done?
	At the time of writing, there was no such existing solution, while mind-mapping was getting more and more popular.
	% How was it done?
	The system was implemented in Scala: Android SDK for client-side and Akka.io for server-side.
